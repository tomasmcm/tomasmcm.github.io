Project Title

Urban Zazen Timer

Description

This iOS mobile app for Zazen Meditation was built using NativeScript Svelte. The UI is build with web technologies and NativeScript is used to load the webview and disable screen sleep while the timer is running.


The UI design is inspired on an existing app, however, it tries to solve a couple of issues namely to follow Zen tradition more closely:

– Available durations changed to be 15, 20, 30 and 40 minutes.
– Interval bells automatically selected based on the duration (none for 15 min, 1 interval for 20 min, and 2 intervals for 30 and 40 min).
– Only 2 clicks to change duration.
– Bell sounds based on sfzc.org audio.


Live app available on the App Store.
