Project Title

Arduinome

Description

Arduinome is a device with a grid of 8 by 8 silicone buttons controlled by an Arduino Leonardo. The Arduino outputs simple MIDI commands via USB making it easy to combine with almost any software that allows MIDI input.

Technologies

Arduino, Processing, SimpleSynth
