Project Title

Revolve.it

Description

This project uses a BBC Micro:bit connected via Bluetooth to a Node.js Server. The name Revolve.it is meant to be a one word instruction manual, since the visualisation is controlled by rotating the physical interface around its centre.


The pitch, tilt, rotation and magnetometer data are used to control an equirectangular video in order to create an immersive 360º experience.


Revolve.it physical interface is a 3D Printed enclosure for the Micro:bit and batteries. The idea behind the design was to have an object whose affordances could unconsciously suggest how it should be held and manipulated, something that would fit into the rested position of one’s hand.

Technologies

BBC Micro:bit, BLE, A-Frame.js, Three.js, Fusion360, Cinema4D
